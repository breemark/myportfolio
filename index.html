Restful API: Braitos

Bug Reporting and Issue Tracker Online System
PHP Laravel 7, Restful API with Json Web Token Authentication.

When working in the Banking Sector, software should be strong to endure the following years. As it is well known, Software in the Banking Industry tends to be used for many years. Security is also a key factor for standards of Banking industry software.

A Restful API can be connected to old systems as well as new software and applications, this flexibility is the key for building scalable systems, no matter the complexity of the existing Software resources.

Restful APIs are independent, which means they will not break existing software, we only need to connect the services in the correct way, by using the HTTP/HTTPS protocol. BRAITOS is a Restful API built in PHP 7.4, Laravel 7 Framework.

BRAITOS is an example of a Server-Side API used to manage information, for any kind of Industry.
